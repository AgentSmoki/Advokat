# ⚖️ Адвокат — первый в РФ Claude Code агент — судебный стратег

![Claude Code](https://img.shields.io/badge/Claude%20Code-agent-blueviolet) ![Юрисдикция](https://img.shields.io/badge/юрисдикция-РФ-blue) ![Регион](https://img.shields.io/badge/регион-Волгоград%20%2B%20РФ-green) ![License](https://img.shields.io/badge/license-MIT-lightgrey)

**Агент-адвокат для [Claude Code](https://claude.com/product/claude-code): изучает материалы дела, находит и проверяет судебную практику, выстраивает правовую позицию и ведёт стратегию всего процесса — от досудебной претензии до кассации.**

Создан на основе исследования российской судебной системы и методологии практикующих судебных юристов. Обкатан на реальном арбитражном деле.

---

## Что умеет

- **7-фазный workflow дела:** карточка дела → правовая квалификация и предмет доказывания → поиск практики → меморандум о перспективах → эшелонированная стратегия → процессуальные документы → сопровождение процесса.
- **Evidence-Anchored цитирование:** каждый судебный акт в документах — только с реквизитами, проверенным первоисточником, цитатой и датой проверки. Ответ на главный риск LLM в юриспруденции: Пленум ВС РФ (2026) обязал раскрывать использование ИИ, за выдуманные нейросетью ссылки на судебные акты уже штрафуют.
- **Иерархия убедительности практики:** КС РФ → Пленум ВС → Обзоры Президиума → коллегии ВС → свой кассационный округ → областной суд → конкретный судья.
- **Юрисдикционный профиль Волгограда из коробки:** вертикали судов (4-й КСОЮ, 12-й ААС, АС Поволжского округа), региональные обобщения практики. Легко адаптируется под любой регион РФ.
- **Красные линии:** запрет выдумывать реквизиты и цитаты, запрет гарантировать исход, напоминание о раскрытии ИИ, финальная проверка каждого документа человеком.
- **Аудит документов против первоисточников:** построчная сверка ссылок на пункты договоров и актов — ловит «фантомные пункты» до того, как их найдёт оппонент.

## Быстрый старт

```bash
git clone https://github.com/AgentSmoki/Advokat.git
cd Advokat
claude
```

Агент активируется автоматически: Claude Code читает `CLAUDE.md` как системную инструкцию. Дальше — просто опишите дело:

```
Изучи дело: [папка с документами]. Мы — ответчик, вот их иск. Построй стратегию.
```

## Структура

```
Advokat/
├── CLAUDE.md      # инструкция агента: роль, workflow, протоколы, границы
├── AGENTS.md      # symlink для cross-tool совместимости (Codex CLI и др.)
├── research/      # исследовательская база: источники практики, суды, статистика
├── docs/          # (локально) планы и материалы дел — в .gitignore
└── memory.md      # (локально) память по делам — в .gitignore
```

## ⚠️ Дисклеймер

Агент — **аналитический инструмент юридической поддержки, а не адвокат и не юридическая услуга**. Каждый документ перед подачей проверяет человек; ответственность за процессуальные решения несёт пользователь. Защиту по уголовным делам осуществляет только адвокат со статусом (ст. 49 УПК РФ). Помните об обязанности раскрывать использование ИИ при подготовке к судебному разбирательству (Постановление Пленума ВС РФ, 2026).
